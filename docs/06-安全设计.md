# 安全设计

## 6.1 认证授权机制

### 6.1.1 Web 端认证
- Session + Cookie 方案
- 过期时间：24 小时
- Cookie 属性：HttpOnly, Secure, SameSite=Strict

### 6.1.2 电视端认证
- Device Token 方案
- UUID v4 作为 device_token
- Android EncryptedSharedPreferences 加密存储

## 6.2 错误码定义

### 通用错误码 (0-999)
| 错误码 | 说明 |
|--------|------|
| 0 | 成功 |
| 500 | 服务器内部错误 |
| 501 | 参数验证失败 |
| 502 | 请求频率过高 |

### 认证错误码 (1000-1999)
| 错误码 | 说明 |
|--------|------|
| 1001 | 用户名或密码错误 |
| 1002 | 账号已被禁用 |
| 1003 | Session 已过期 |
| 1004 | 未登录 |
| 1005 | 权限不足 |

### 设备错误码 (2000-2999)
| 错误码 | 说明 |
|--------|------|
| 2001 | 绑定码无效或已过期 |
| 2002 | 绑定码已被使用 |
| 2003 | 设备已绑定其他账号 |
| 2004 | 设备未绑定 |
| 2005 | 设备 Token 无效 |

### 书籍错误码 (3000-3999)
| 错误码 | 说明 |
|--------|------|
| 3001 | 文件格式不支持 |
| 3002 | 文件解析失败 |
| 3003 | 文件过大（限制 50MB） |
| 3004 | 书籍不存在 |
| 3005 | 无权访问此书籍 |

### 阅读控制错误码 (4000-4999)
| 错误码 | 说明 |
|--------|------|
| 4001 | 已达每日阅读时长限制 |
| 4002 | 已达连续阅读时长限制 |
| 4003 | 当前为禁止阅读时段 |

## 6.3 数据安全

- 密码使用 bcryptjs 加密，10 轮 salt
- 文件类型白名单：EPUB, PDF, TXT
- 文件大小限制：50MB
- 所有 API 强制 HTTPS
- SQL 注入防护（参数化查询）
