# 接口设计

## 4.1 API 接口总览

|------|---------|---------|
| /api/auth | 3 | 登录、登出、Session 检查 |

| /api/devices | 5 | 设备绑定、管理、远程控制 |
| /api/books | 8 | 上传、解析、管理、授权、预览 |

**总计：39 个接口**

## 4.2 认证模块 API

### POST /api/auth/login
**功能：** 管理员登录

**请求：**
```json
{
  "username": "admin",
  "password": "password"
}
```

**响应：**
```json
{
  "code": 0,
  "data": {
    "admin_id": 1,
    "username": "admin",
    "email": "admin@example.com"
  }
}
```

**错误码：**
- 1001: 用户名或密码错误
- 1002: 账号已被禁用

### POST /api/auth/logout
**功能：** 管理员登出

### GET /api/auth/session
**功能：** 检查 Session 有效性
