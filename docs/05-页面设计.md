# 页面设计

## 5.1 Web 端页面设计（9个页面）

1. **登录页面 (Login.tsx)** - 用户名密码登录
2. **仪表盘页面 (Dashboard.tsx)** - 今日/本周统计、实时阅读状态
3. **子账号管理页面 (Children.tsx)** - 子账号CRUD
4. **设备管理页面 (Devices.tsx)** - 设备绑定、远程控制
5. **书籍管理页面 (Books.tsx)** - 书籍列表、上传
6. **书籍详情页面 (BookDetail.tsx)** - 书籍信息、授权管理
7. **阅读管控配置页面 (Control.tsx)** - 防沉迷策略配置
8. **阅读统计页面 (Stats.tsx)** - 阅读数据统计
9. **书签管理页面 (Bookmarks.tsx)** - 书签列表管理

## 5.2 电视端页面设计（4个页面）

1. **设备绑定页面 (BindFragment.kt)** - 显示绑定码
2. **书架页面 (ShelfActivity.kt)** - 书籍列表展示
3. **阅读页面 (ReaderFragment.kt)** - 文本阅读、翻页
4. **锁屏休息页面 (LockFragment.kt)** - 防沉迷锁屏
